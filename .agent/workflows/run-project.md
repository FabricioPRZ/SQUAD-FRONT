---
description: Cómo correr el proyecto SQUADUP localmente
---

Para ejecutar la aplicación en tu entorno local, sigue estos pasos:

1. **Instalar dependencias** (solo la primera vez o si se agregan nuevas librerías):
   ```bash
   npm install
   ```

// turbo 2. **Iniciar el servidor de desarrollo**:

```bash
npm run start
```

3. **Acceder a la aplicación**:
   Abre tu navegador en [http://localhost:4200](http://localhost:4200)

4. **Vistas principales disponibles**:
   - Landing: `/`
   - Login: `/login`
   - Registro: `/register`
   - Dashboard: `/dashboard`
   - Home: `/home`
